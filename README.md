# AWS-Terraform
Example project to deploy an AWS architecture with Terraform

## Steps to make a Terraform / Ansible instalation for AWS
* Enter in root mode
  ```console
  user@domain:~$ sudo su
  ```
* Check that Python was intalled
  ```console
  root@domain:~$ python --version
  ```
* Make an update
  ```console
  root@domain:~$ apt-get update
  ```
